Component docs, review workflow, and tracker for Pulse — replacing Supernova.

- **Draft & review**: edit pages at `/admin`, comments happen via PR (or directly if you're logged in through Netlify Identity).
- **Tracker fields**: every component page carries `status`, `owner`, and `trackerUpdated` in its frontmatter — see the [Components](/components/button/) section.
- **Source of truth**: this GitHub repo. What's published here is what's shipped.
