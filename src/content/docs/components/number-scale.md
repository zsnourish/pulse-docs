**Number scale component** allows users to select a numeric value from a predefined linear range (e.g. 1-5 or 1-10). Often used for ratings or Likert scales (e.g. Strongly Disagree to Strongly Agree).

## Usage & Content Guidelines

<div class="ds-callout ds-do">
<p><strong>✓ Do</strong> — Use for quick, subjective feedback like satisfaction or agreement ratings.</p>
</div>
<div class="ds-callout ds-do">
<p><strong>✓ Do</strong> — Clearly label the meaning of the scale endpoints (e.g. Poor to Excellent).</p>
</div>
<div class="ds-callout ds-dont">
<p><strong>✗ Don't</strong> — Do not use for precise data input where a user might need to enter a specific number.</p>
</div>
<div class="ds-callout ds-dont">
<p><strong>✗ Don't</strong> — Avoid a scale with more than 10 points.</p>
</div>
