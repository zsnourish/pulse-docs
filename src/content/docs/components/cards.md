**Card components** are containers for sets of related information and actions, and can also contain panels e.g. for lists. Cards can be static, or interactive and clickable.

### Usage & Content Guidelines

<div class="ds-callout ds-do">

</div>

</div>
<div class="ds-callout ds-dont">

</div>

## Empty States

Card component variant used when there is no data to display, providing context and a guided action to the user. Includes an icon or illustration, a short message and where possible, a primary action button (e.g., "Add Item") to guide the user's journey.
