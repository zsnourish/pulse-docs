**Dropdown component** displays a list of options from which a user can select a single choice. Supports simple lists, lists with grouping including a search bar to filter long lists. Selection replaces the default label.

## Usage & Content Guidelines

<div class="ds-callout ds-do">
<p><strong>✓ Do</strong> — Use groupings and search for lists with 15+ items to improve scannability.</p>
</div>
<div class="ds-callout ds-dont">
<p><strong>✗ Don't</strong> — Avoid using for a binary choice or fewer than 5 common options; consider radio buttons instead.</p>
</div>
<div class="ds-callout ds-dont">
<p><strong>✗ Don't</strong> — Do not make the list excessively long without search or groupings.</p>
</div>
