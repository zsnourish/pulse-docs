**Three dot menu component** is a vertical ellipsis icon that toggles a contextual menu of secondary actions. The menu is typically attached to a specific item (e.g. a table row or card).

### Usage & Content Guidelines

<div class="ds-callout-columns">
<div class="ds-callout-col">
<div class="ds-callout ds-do">
<p><strong>✓ Do</strong> — Use for secondary, tertiary or destructive actions to reduce UI clutter.</p>
</div>
<div class="ds-callout ds-do">
<p><strong>✓ Do</strong> — Label menu items clearly with text and icons.</p>
</div>
</div>
<div class="ds-callout-col">
<div class="ds-callout ds-dont">
<p><strong>✗ Don't</strong> — Do not hide primary user actions in this menu.</p>
</div>
