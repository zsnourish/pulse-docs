**Checkbox component** allows the user to select one or multiple items from a list. For a single checkbox, it represents a binary 'on/off' state. In a group, multiple options can be selected.

## Usage & Content Guidelines

<div class="ds-callout ds-do">
<p><strong>✓ Do</strong> — Use for multiple selection or for a single binary agreement (e.g., 'I agree to the terms').</p>
</div>
<div class="ds-callout ds-do">
<p><strong>✓ Do</strong> — Group related checkboxes together with a clear legend or label.</p>
</div>
<div class="ds-callout ds-dont">
<p><strong>✗ Don't</strong> — Do not use for mutually exclusive options; use Radio Buttons instead.</p>
</div>
<div class="ds-callout ds-dont">
<p><strong>✗ Don't</strong> — Avoid using a single checkbox to perform an immediate action; use a toggle switch instead.</p>
</div>
